Debian Team Activity Metrics Prototype

Static Plot Display

Access URL : http://debmetrics.appspot.com/<team>/<metrics>/
Example http://debmetrics.appspot.com/teammetrics/commitlines/

Dynamic Plot Rendering

Access URL : http://debmetrics.appspot.com/dynamic/<team>/<metrics>/
Example http://debmetrics.appspot.com/dynamic/teammetrics/commitlines/

Data API

Access URL : http://debmetrics.appspot.com/api/v1/<team>/<metrics>/
Example http://debmetrics.appspot.com/api/v1/teammetrics/commitlines/




The images and the data is scraped live from Debian servers and they hold the copyright of the data. This website is a Free Software and the source is available on github. Please feel free to fork and add to it.
